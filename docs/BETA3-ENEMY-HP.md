# Beta-3: 적 캐릭터 실시간 HP 바 (Enemy Health Bars)

## 목표
플레이어가 각 적 캐릭터의 남은 체력을 직관적으로 확인하고 전략적인 전투를 수행할 수 있도록 모든 적 캐릭터 위에 실시간 HP 바를 추가합니다.

## 상세 구현 내용
1. **베이스 클래스 연동 (`lib/components/enemy.dart`)**
   - 모든 적의 조상인 `Enemy` 클래스에 `HpBarComponent`를 추가했습니다.
   - 적이 생성될 때(`onLoad`) 캐릭터 바로 위에 체력 바가 나타나도록 설정했습니다.
   - 데미지를 입을 때(`takeDamage`) 자동으로 바 수치가 동기화됩니다.

2. **캐릭터별 커스텀 조정**
   - **골렘 (`lib/components/golem.dart`)**: 일반 적보다 체력이 높고 덩치가 크기 때문에, HP 바의 너비를 1.2배 넓게 설정하고 위치를 보정했습니다.
   - **궁수 (`lib/components/archer.dart`)**: 베이스 클래스의 설정을 그대로 따르며, 일관된 시각 언어를 유지합니다.

3. **시각적 특징**
   - 플레이어 HP 바와 동일한 색상 체계를 사용합니다 (초록 -> 오렌지 -> 빨강).
   - 캐릭터 이동 시 부모 컴포넌트를 따라 함께 이동합니다.

## 수용 기준
- [x] 모든 종류의 적 머리 위에 HP 바가 표시됨.
- [x] 데미지 입을 시 바의 길이가 즉각적으로 줄어듬.
- [x] 적 사망 시 HP 바도 함께 소멸됨.

## 향후 계획
- 특정 보스 캐릭터 등장 시 화면 상단에 거대한 전용 HP 바를 표시하는 기능을 검토 중입니다.
