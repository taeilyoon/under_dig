# Alpha-2-13: 아이템 사용 UI/UX 설계 및 구현 (Item Usage UI/UX)

## 목표
- 플레이어가 인벤토리에 보유한 아이템을 확인하고 전략적으로 사용할 수 있는 직관적인 HUD(Heads-Up Display) 인터페이스를 구축합니다.
- 아이템 사용 시 즉각적인 시각적 피드백을 제공합니다.

## HUD 레이아웃 디자인
- **위치**: 화면 하단 중앙 또는 우측 하단에 고정된 가로형 바.
- **슬롯 구성**: 최대 5개의 슬롯을 아이콘 형태로 표시.
- **상태 표시**: 비어 있는 슬롯은 반투명하게, 아이템이 있는 슬롯은 해당 아이템의 대표 아이콘과 개수(중첩 시) 표시.

## 사용자 인터랙션 (UX)
1. **아이템 선택**: 슬롯을 클릭하거나 터치하면 해당 아이템의 툴팁이 나타납니다.
2. **아이템 사용**: 툴팁 내의 '사용' 버튼을 누르거나, 슬롯을 더블 탭하여 즉시 사용합니다.
3. **툴팁 정보**: 아이템 이름, 효과 설명(예: "+10 HP 회복"), 지속 시간 정보를 포함합니다.

## 시각적 피드백
- **아이템 사용 시**: 슬롯이 반짝이며 사라지는 애니메이션 효과.
- **인벤토리 가득 참**: 새로운 아이템 획득 시 인벤토리가 가득 찼다면 화면 상단에 경고 메시지 표시.
- **버프 활성화**: 아이템 사용 후 버프가 활성화되면 캐릭터 주변이나 HUD 상단에 해당 버프 아이콘 표시.

## 구현 계획
- `lib/ui/inventory_hud.dart`: Flame 또는 Flutter 위젯 기반의 인벤토리 HUD 컴포넌트입니다.
- `lib/ui/item_tooltip.dart`: 아이템 상세 정보를 보여주는 팝업/오버레이 컴포넌트입니다.
