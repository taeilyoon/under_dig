# Beta-3-05: 액션 카드 시스템 설계 및 프로토타이핑 (Action Card System)

## 목표
- 플레이어가 매 턴 또는 특정 시점에 선택하여 사용할 수 있는 '액션 카드' 시스템을 도입하여 로그라이트 요소와 전략적 선택의 재미를 강화합니다.
- 단순 이동/공격 외의 특수 능력을 카드를 통해 발동합니다.

## 시스템 개요
1. **카드 획득 (Deck Building)**
   - 스테이지 클리어 보상 또는 상점(추가 예정)에서 새로운 카드를 선택하여 덱에 추가합니다.
2. **카드 사용 (Action)**
   - 사용 가능한 카드 중 하나를 선택하여 발동합니다. 
   - 사용 시 특정 자원(예: 행동 포인트, 쿨다운)을 소모할 수 있습니다.
3. **카드 종류 (Card Types)**
   - **공격형**: 전방 3칸 공격, 관통 공격 등.
   - **유틸리티**: 위치 바꾸기, 순간 이동, 블록 파괴.
   - **방어형**: 일시적 무적, 밀쳐내기.

## 데이터 모델 설계
- `ActionCard`: 카드의 이름, 설명, 이미지, 효과 목록.
- `ActionEffect`: 카드 발동 시 발생하는 실제 물리적/상태적 변화.

## 인벤토리와의 차이
- **아이템**: 필드에서 획득하는 일회성 소모품.
- **액션 카드**: 플레이어의 영구적인 스킬셋에 가깝지만, 무작위로 뽑혀 사용되는 시스템.

## 구현 계획
- `lib/models/card_model.dart`: 카드 데이터 구조 정의.
- `lib/managers/card_manager.dart`: 플레이어의 덱(Deck)과 핸드(Hand) 관리.
- `lib/ui/card_hand_hud.dart`: 화면 하단에 현재 사용 가능한 카드들을 나열하는 UI.

## 향후 작업
- 카드 등급(Normal, Rare, Epic) 시스템 도입.
- 카드 강화(Level Up) 시스템 연동.
